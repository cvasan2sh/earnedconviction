# earnedconviction.com — Cowork Context

## What this is
Personal brand and publishing platform for Siva Pentakota.
Built on The Forge — a multi-agent deliberation system.
Live at earnedconviction.com.

## Tech stack
Next.js App Router, TypeScript, Tailwind CSS
Vercel hosting, Cloudflare DNS, Resend newsletter

## Design system — never violate
- Background: #1A1A1A
- Text: #F5F0E8  
- Accent: #C0392B
- Display font: Playfair Display
- Body font: Source Serif 4
- No hero sections, no card grids, no purple gradients

## Key file locations
- Pages: src/app/[page]/page.tsx
- Page content: src/content/pages/[page].mdx
- Articles: src/content/writing/
- Projects data: src/data/projects.ts
- Forge sessions: forge-sessions/
- Components: src/app/components/ and src/components/
